# Videofy

A front-end focused web app inspired by YouTube that I developed. Includes features such as infinite scrolling to resemble a realistic video feed, OAuth-based authentication, and more.

Stack used:

- React
- Redux
- YouTube API
- Semantic UI
- SCSS
